# Cybersecurity Portfolio

## Overview

This repository contains hands-on cybersecurity and network security labs completed in a virtual lab environment using Cisco Packet Tracer, VirtualBox, Ubuntu Linux, Kali Linux, and various Linux networking and security tools.

The portfolio focuses on:
- enterprise network segmentation
- Linux networking
- attack simulation
- log analysis
- troubleshooting
- security investigation techniques

The goal of these labs is to develop practical cybersecurity, networking, and SOC-related skills through real configuration, testing, and analysis.

---

# Labs Included

## 1. Enterprise Network Segmentation Lab

Built a segmented enterprise-style network using Cisco Packet Tracer.

### Topics Covered
- VLAN configuration
- Inter-VLAN routing
- Router-on-a-stick
- 802.1Q trunking
- Static IP configuration
- Connectivity troubleshooting
- VirtualBox internal networking
- Linux networking with Netplan

### Technologies Used
- Cisco Packet Tracer
- Ubuntu Linux
- VirtualBox

### Skills Demonstrated
- Network segmentation
- Switch configuration
- Router configuration
- Linux network administration
- Troubleshooting connectivity issues

## 2. SSH Brute Force Attack & Log Analysis

Simulated an SSH brute-force attack in a controlled lab environment and analysed authentication logs from a defender’s perspective.

### Topics Covered
- SSH configuration
- Hydra brute-force simulation
- Authentication log analysis
- Failed login investigation
- Successful authentication tracking

### Technologies Used
- Kali Linux
- Ubuntu Linux
- Hydra
- OpenSSH

### Skills Demonstrated
- Attack simulation
- Log analysis
- Linux administration
- SSH troubleshooting
- Security monitoring
